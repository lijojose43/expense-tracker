Use this guide to add Expense Tracker to your phone or desktop for faster launch, offline support, and a cleaner full-screen experience.

AN Android

  1. Open the app URL in Chrome or Edge.
  2. Tap menu (three dots), then select "Install App" or "Add to Home Screen".
  3. Confirm install and launch from your home screen.

IO iOS (Safari)

  1. Open the app URL in Safari.
  2. Tap Share, then choose "Add to Home Screen".
  3. Tap Add, then launch the new icon from the home screen.

PC Desktop

  1. Open the app in Chrome, Edge, or Brave.
  2. Click the install icon in the address bar (or browser menu install option).
  3. Confirm to install and open from applications/start menu.

Troubleshooting

  • Use HTTPS and wait for the page to finish loading before trying install.
  • If install option does not appear, refresh and clear browser cache.
  • Open online once first, then test offline usage.
  • If icon looks wrong, reinstall after clearing cached site data.

Pre-Install Checklist

  • App opens without errors
  • Add/edit/delete transactions works
  • Theme switching works
  • Data persists after refresh
